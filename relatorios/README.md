# Relatórios (repo Aworklanza)

Saídas geradas pela CLI em `src/run.ts`. **Prestação de contas para parceiros** (PDF/planilha mensal) continua no Dropbox (`Financeiro/prestação de contas/`).

| Pasta | Conteúdo |
|-------|----------|
| `quebra-contrato/` | Acerto de encerramento — `quebra-contrato-*.txt` (documento para o cliente) |
| `prestacao-contas/` | Relatórios internos do repo (ex. listagens semanais Rastreame) |
| `sync/` | Lotes DETRAN — `_sync_infracoes.json`, `_sync_ipva_licenciamento.json` |
| `_tmp/` | Rascunhos e JSON temporários (`_*`) |

Novos relatórios passam a gravar nestas subpastas automaticamente.
